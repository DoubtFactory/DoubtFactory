# Doubt Factory Changelog

## v1.5.0
- Improved mobile responsiveness across the home, question, search, and admin pages
- Enhanced touch targets, spacing, typography, and keyboard accessibility
- Added clearer focus states, labels, and ARIA-friendly feedback for forms and search
- Improved performance by reducing unnecessary DOM updates and using document fragments
- Added lazy loading for embedded video content and better empty/loading states
- Polished navigation, semantic structure, meta tags, and canonical links for launch readiness

## v1.4.0
- Added a more professional admin dashboard experience
- Introduced a sidebar-based layout with mobile-friendly behavior
- Improved form organization with clearer cards and sections
- Added a live preview panel for question creation
- Added autosave draft support in localStorage
- Added form validation and toast-style notifications
- Added loading feedback while saving and better delete confirmation

## v1.3.0
- Added a more professional and responsive search experience
- Introduced instant search with debounce input
- Added search suggestions and keyboard navigation support
- Added filter chips for exam and difficulty-focused filtering
- Improved empty states, loading feedback, and result card presentation

## v1.0.0-alpha.1
- Project started
- Requirements finalized
- Development structure created
