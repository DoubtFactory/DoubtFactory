# Doubt Factory Changelog

## Version 1.6
- Analytics Dashboard for the admin experience
- Added dashboard statistics for questions, subjects, chapters, exams, videos, and comments
- Added recent activity feed using the existing question data
- Added quick action cards for adding, managing, browsing, and searching questions
- Added lightweight analytics sections for questions by subject and exam
- Improved responsive admin layout and overview experience

## Version 1.5
- Professional Manage Questions dashboard experience
- Better table presentation with stronger visual hierarchy
- Added instant search, filtering, sorting, and pagination
- Added loading state, empty state, and responsive dashboard layout
- Improved management workflow with preview, edit, and delete actions
- Added toast notifications and a delete confirmation modal

## v1.4.0
- Added a more professional admin dashboard experience
- Introduced a sidebar-based layout with mobile-friendly behavior
- Improved form organization with clearer cards and sections
- Added a live preview panel for question creation
- Added autosave draft support in localStorage
- Added form validation and toast-style notifications
- Added loading feedback while saving and better delete confirmation

## v1.3.0
- Added a more professional and responsive search experience
- Introduced instant search with debounce input
- Added search suggestions and keyboard navigation support
- Added filter chips for exam and difficulty-focused filtering
- Improved empty states, loading feedback, and result card presentation

## v1.0.0-alpha.1
- Project started
- Requirements finalized
- Development structure created
